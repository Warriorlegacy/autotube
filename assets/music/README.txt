Background Music for AutoTube Videos

Place CC0 / royalty-free MP3 files in this directory.
The first file (alphabetically) will be used as default background music.

Sources for CC0 music:
- YouTube Audio Library (https://studio.youtube.com/channel/UC/music)
- Pixabay Music (https://pixabay.com/music/)
- Free Music Archive (https://freemusicarchive.org/)
- Incompetech (https://incompetech.com/) — requires attribution

Each track added here must be CC0 or have a compatible license.
Add attribution text next to each file's metadata.
